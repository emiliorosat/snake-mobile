# Dr. Snake App Mobile

### Habilidades Ejercitadas

- [IONIC]
- [Angular]
- [Typescript]

## Probar
  1. clonar repositorio
  ```bash 
  git clone "uri repo"
  ```
  2. Acceder al directorio
  ```bash
  cd snake-mobile
  ```
  3. Instalar dependencias
  ```bash
  npm install
  ```
  4. Probar
  ```bash
  ionic serve --lab
  ```
